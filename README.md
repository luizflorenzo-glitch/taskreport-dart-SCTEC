# TaskReport Dart

## Sobre o projeto

TaskReport Dart é um simulador de tratamento de dados de tarefas desenvolvido em Dart.

O projeto simula o recebimento de dados de uma API no formato de lista de mapas, transforma esses dados em objetos, trata inconsistências como campos nulos e valores em formato texto, e gera um relatório consolidado no console.

## Objetivo

Praticar os conceitos do Módulo 01 do curso de Desenvolvimento Mobile FLUTTER:

- Lógica de programação
- Linguagem Dart
- Tipos de dados: String, int, double, bool, List, Map, Set
- Estruturas de decisão e repetição
- Funções e arrow functions
- Métodos de coleção: map, where, reduce
- Classes, objetos, atributos e métodos
- Construtores e encapsulamento
- Herança e polimorfismo com @override
- GitHub, GitFlow e Kanban
